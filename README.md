1. Крайне рекомендуется клонить с `git clone --depth 1 git@github.com:zemlianin/uav.git`, потому что мы зачем-то хранили кучу бинарей...
2. Датасет с Roboflow нужно положить в папку `datasets`:
```
.
├── datasets
│  ├── data.yaml
│  ├── test
│  ├── train
│  └── valid
├── ...
├── README.md
├── ...
└── yolov8n.pt
```
